Terminology

Alphabetical reference for the terms used in this proposal.

Adjusted Evidence

Useful evidence after quality, freshness, duplication, and concentration adjustments.

Adjusted Pool Rank Score

The placement score after pool fit and repeat control are applied.

Attention

Current user attention, based on fresh and meaningful engagement.

Base Cultural Score

The weighted cultural relevance value before multipliers are applied.

Block

An eligibility status that stops a trend from public discovery.

Breadth

How widely interest spreads across users, communities, and sources.

Candidate Pool

A group of trends competing for feed space for the same reason.

Candidate Pool Assignment

The stage that assigns eligible trends to one or more internal pools.

Cap

An eligibility status for valid trends that should have limited distribution.

Cap Availability

Whether a trend still has exposure available under its cap ceiling.

Cap Ceiling

The strongest active exposure limit for a trend.

Caps

Feed limits that stop valid but imperfect trends from taking too much space.

Confidence Multiplier

A bounded multiplier that lowers ranking strength when evidence is thin.

Constraint

A rule that limits placement, exposure, repetition, or another feed behavior.

Content Availability

Whether the app can still render the content needed to serve an item.

Content Gaps

Missing media, sources, summary, explanation, or display coverage.

Context Gap

The missing context readiness, calculated from Context Sufficiency.

Context Sufficiency

Whether the trend has enough explanation, sources, and framing to make sense.

Coordination Risk

The likelihood that activity is coordinated rather than organic.

Coverage Multiplier

A bounded modifier based on how much usable display material exists.

Coverage Sufficiency

Whether NOISE has enough usable display options for the trend.

Curation Authority

The contribution of trusted curator judgment and review signals.

Curation Opportunity

A priority score for culturally relevant trends that need better content.

Curator Explanation

Curator reasoning that explains why a trend matters.

Curator Queue

The route for trends that need curator review or improvement.

Curator Reward Score

A score for useful curator work after the work is done.

Deduplication

A rule that prevents the same trend from appearing twice in one feed window.

Definition Quality

How clearly and specifically the trend is defined.

Discussion Quality

The usefulness of posts, comments, sources, and explanations around a trend.

Display Readiness

The best available UI representation for a trend.

Duplicate Check

A serving check that prevents repeating the same Trend ID.

Duplicate Reversals

A penalty input for curator work reversed because it duplicated existing content.

Early / Niche

A pool for promising trends with focused early interest.

Editorial Usefulness

Whether the trend entry is understandable and useful to users.

Eligibility Routing

The first routing stage: Block, Hold, Cap, or Pass.

Eligibility Status

The status assigned by Eligibility Routing.

Evidence Quality

How credible, relevant, fresh, and sufficient the evidence is.

Evidence Target

The evidence level where confidence is considered mature for that category.

Exploration

Controlled feed space for promising but under-tested trends.

Falling Fast

A pool for trends losing relevance quickly in a meaningful way.

Feed Re-ranking & Mixing

The stage that turns scored candidates into a balanced feed window.

Feed Serving

The final step that returns the next valid item to the app.

Feed Window

The internal set of feed positions being planned at once.

Feed Window Size

The number of feed positions being planned in a window.

Final Candidate Queue

The ordered list produced before items are served.

Flags / Reversals

A reward penalty for moderation flags, duplicate reversals, low-effort uploads, or misleading context.

Formula

A calculation that combines inputs into a new value.

Hard Display Issue

A display problem serious enough to block a public surface.

Hold

An eligibility status for trends that need review or preparation.

Integrity Limit

An exposure limit that gets stricter as manipulation risk rises.

Integrity Multiplier

A multiplier that reduces ranking strength when manipulation risk exists.

Low-Effort Uploads

A penalty input for shallow or barely useful curator submissions.

Main Swipe Feed

The highest-visibility one-at-a-time discovery surface.

Manipulation Risk

A normalized risk score for artificial or suspicious activity.

Market Conviction

The strength and reliability of buy/short behavior.

Metric

A measured or calculated value used by a formula or rule.

Misleading Context

A penalty input for inaccurate or unsupported curator framing.

Moderation Flags

A penalty input for safety, policy, or trust issues.

Momentum Direction

Whether movement is rising, falling, or flat.

Momentum Intensity

How unusually fast relevance is moving.

Multiplier

A factor that scales ranking strength.

Needs Better Content

A pool for important trends that need better presentation or context.

New Quality Value

The quality, presentation, or context level after curator work.

Niche Intensity

How strongly a specific community or subculture cares about a trend.

NOISE Relevance Score

The ranking score used to compare eligible trends.

Open Slot Budget

The remaining room a pool has in the current feed window.

Pass

An eligibility status for trends that clear the first routing check.

Personal

A pool for trends with strong user affinity.

Placement Role

The one pool label a trend occupies in a specific feed window.

Placement Role Selection

The rule that chooses one placement role for a multi-pool trend.

Pool Allocation

The process of assigning feed slots to pool reasons using target weights.

Pool Fit

How strongly a trend matches a specific candidate pool.

Pool Rank Score

The score used to order candidates inside one placement role.

Presentation Gap

The missing presentation readiness.

Presentation Multiplier

A bounded multiplier based on Presentation Readiness.

Presentation Readiness

Whether NOISE has a strong visual or text display path.

Previous Quality Value

The quality, presentation, or context level before curator work.

Quality Improvement

The before-and-after improvement created by curator work.

Quality Limit

An exposure limit based on quality or presentation weakness.

Quality Multiplier

A bounded ranking adjustment for valid trends with quality issues.

Quality Score

The 0-100 quality value used to calculate the Quality Multiplier.

Rank Inside Pool

The rule that orders candidates within a selected pool.

Raw Confidence

The evidence maturity value before the bounded Confidence Multiplier.

Readiness Gap

The largest missing readiness area: presentation or context.

Relevance Scoring

The stage that calculates the NOISE Relevance Score.

Repeat Control

A rule that spaces out similar or recently seen items.

Repeat Multiplier

A multiplier that lowers ranking strength for repetitive items.

Review Approval

A reward input for curator work approved as useful.

Reward Factor

A component that increases or reduces the curator reward calculation.

Rising Fast

A pool for trends gaining relevance quickly.

Route Set

The list of surfaces or work queues selected by a routing step.

Rule

A condition or constraint that changes routing, placement, or exposure.

Search / Detail

A lower-risk surface where users can inspect context.

Serving Checks

Final pass/skip checks before the app receives the next feed item.

Serving Feedback Log

The event log created after serving an item.

Signal

Evidence used by scoring or routing.

Slot Budget

The number of feed positions available for a pool.

Source Context

Whether sources support the trend explanation.

Spam Risk

The likelihood that activity is low-quality, automated, or abusive.

Stage

A step in the algorithm pipeline with a specific job.

Status

A categorical output assigned by a routing step.

Status Limit

An exposure limit based on Eligibility Status.

Successful Publication

A reward input for work that unlocks a surface or stronger distribution.

Summary Usefulness

Whether the short explanation helps users understand the trend.

Surface Routing

The stage that decides where a trend can appear or where it needs work.

Target Slot Weights

Tunable weights that control each pool's share of the feed window.

Text Readiness

Whether NOISE can present the trend well through text.

Title Clarity

Whether the trend name is specific, readable, and unambiguous.

Top Now

A pool for the strongest current cultural trends.

User Affinity

How strongly a trend matches a user's interests or behavior.

User Response

A reward input based on post-publication engagement.

Visual Readiness

How well images or video can represent the trend.

Wash-Trading Risk

The likelihood that market activity is artificially created.

Whale Concentration

How much activity depends on a small number of large actors.
